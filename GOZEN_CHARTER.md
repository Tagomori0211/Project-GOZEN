# 🏯 Project GOZEN 御前会議

## ~海軍と陸軍の建設的対立で、最高のエンジニアリング決定を導く~

---

## 概要

このプロジェクトは、複数のAIエージェント（Claude × Gemini）を
**異なる「立場」に配置することで、自動的に「相互監視」を実現するシステム**です。

名前の由来は **「御前会議（ごぜんかいぎ）」**。

天皇の面前で行われた政策決定会議のように、
このシステムでも **国家元首（人間）** の前で、
海軍参謀（Claude）と陸軍参謀（Gemini）が
激論を交わし、最高の決定が導き出されます。

---

## なぜ「対立」が必要なのか

AI開発における最大の敵は **「グループシンク」** です。
複数のAIが全員「Yes」と言うと、誰も間違いを指摘しません。

このシステムでは：

- 🌊 **海軍参謀（Claude）** = 理想・論理・スケーラビリティ重視
- 🪖 **陸軍参謀（Gemini）** = 現実・実装・運用負荷重視

この2つの正義が対立することで、自動的に「最適な折衷案」が生まれます。

## 公式モットー

## 使い方

```bash

python gozen/cli.py --mode sequential task.yaml

python gozen/cli.py --mode parallel --plan max5x task.yaml
```
